# 🌦️ Weather App

A simple and responsive Weather Application built using **HTML, CSS, and JavaScript**. The app fetches real-time weather information for any city using the **WeatherAPI** and displays the current temperature, weather condition, and weather icon.

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Displays current temperature
- ☁️ Shows weather condition
- 🖼️ Displays weather icon
- ❌ Error handling for invalid city names
- 📱 Responsive and clean user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- WeatherAPI

## ⚙️ How to Run

1. Clone the repository.

```bash
git clone https://github.com/your-username/weather-app.git
```

2. Open the project folder.

```bash
cd weather-app
```

3. Open `index.html` in your web browser.

No additional installation is required.

## 🔑 API

This project uses the **WeatherAPI** to fetch live weather data.

You can get your free API key from:
https://www.weatherapi.com/

Replace the API key in the JavaScript code:

```javascript
const apiKey = "YOUR_API_KEY";
```
